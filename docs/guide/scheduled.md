# 定时公告推送

Cron 表达式驱动，每天/每小时自动推送公告。

## 配置项

```lua
ScheduledPush = {
    Enable = true,
    Jobs = {
        {
            Name = "早安公告",
            Cron = "0 9 * * *",       -- 每天早上9点
            Channel = "napcat",
            Message = "🌅 早上好！今日服务器运行良好！",
            Enabled = true,
        },
        {
            Name = "晚安公告",
            Cron = "0 23 * * *",      -- 每天晚上11点
            Channel = "napcat",
            Message = "🌙 各位玩家晚安，明天再见！",
            Enabled = true,
        },
    },
}
```

## Cron 表达式

```
分 时 日 月 周
│  │  │  │  │
│  │  │  │  └─── 0=周日，1=周一... 6=周六，* = 每天
│  │  │  └────── 月（1-12，* = 每月）
│  │  └───────── 日（1-31，* = 每天）
│  └──────────── 时（0-23）
└──────────────── 分（0-59）
```

## 常用示例

| 表达式 | 说明 |
|--------|------|
| `0 9 * * *` | 每天早上9点 |
| `0 12 * * 6` | 每周六中午12点 |
| `0 */2 * * *` | 每隔2小时 |
| `0 9,18 * * *` | 每天9点和18点 |
