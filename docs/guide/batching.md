# 消息聚合

短时间多条玩家进出消息合并成一条推送，减少刷屏。

## 聚合规则

- **仅作用于玩家进出事件**
- TX Admin / 心跳 / 定时公告**不受聚合影响**，始终实时推送

## 配置项

```lua
Batching = {
    Enable = true,            -- 是否开启聚合
    MaxMessages = 10,         -- 积压达到这个数量立即发送
    FlushIntervalMs = 600000, -- 等待多久后发送（毫秒），默认10分钟
    MaxBatchSize = 50,        -- 单次聚合最大条数
}
```

## 建议

- 服务器人数多 → 调小 `FlushIntervalMs`（如 5 分钟）
- 服务器人数少 → 保持默认 10 分钟
